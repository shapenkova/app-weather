# **Vue Weather** 
## Как запустить?

1. Склонировать репозиторий к себе на ПК
2. npm install
3. npm run dev
4. Готово!

## Описание проекта:

Приложение для просмотра погоды App-Weather.

## Стэк технологий:
- Vue 3
- Vite
- Options API
- Prettier
